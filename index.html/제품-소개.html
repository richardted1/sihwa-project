SS41, SS400 /  S45C (슬라브 절단)

단순 절단 :
 T : 두께 (최대 350), W : 폭 (최대2,438), L : 길이 (최대 6,096mm)
 
모형 절단 :
 T : 두께 (최대 350), W : 폭 (최대2,438), L : 길이 (최대 6,096mm)

주 문 형 식 : 
재질(SS400 / S45C 등) 선택 → 두께(T) x 폭(W) x 길이(L) - 수량(N)
※모형 절단 발주 시, 도면 첨부

납 기 :
1일 (소량 발주 시)
※ 다량이거나 가공품일 시 3~4일 (사무실로 문의)

제품 특징 :
• 소 품종 대량 생산  Mass-produced goods
• 직각 및 수평 정밀도 ​Right angle and horizontal precision
• NAMASHI 및 열처리 가능 Possible the heat treatment
• 생산 공정 과정 축소 및 유휴비용 절감 ​Reduce idle costs.
• 신속한 납기 Prompt Delivery
